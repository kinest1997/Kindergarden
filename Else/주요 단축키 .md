# 주요 단축키 
| 이름 | 단축키 | 비고  |
| --- | --- | ---|
| Quick Help | `command + shift + O` | 알파벳 o |
| apple dev document | `command + shift + 0` | 숫자 0 |
| 자동완성 켜기 | `esc` |
| 자동정렬  | `ctrl + I `| 마법
| 특정줄 이동 | `cmd + L`   | `command + F` 의 줄 버전
| 빌드 하기 | `cmd + R`|
| 함수 또는 마크별 이동 | `ctrl + 6 `
| 단어 단위로 움직이기 | `option + 왼쪽/오른쪽`
| 선택한부분 통째로 움직이기 | `cmd + opt + [ or ]` | 이거찾는데 1시간걸림
